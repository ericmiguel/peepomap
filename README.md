# Pepomap

Just some extra Matplotlib colormaps.

## Installation

```bash
pip install pepomap
```

## Colormaps

```python
import pepomap

pepomap.tools.display_colormaps(pepomap.cmaps)
```

```python
import pepomap

cmap = pepomap.cmaps["storm"]
```
